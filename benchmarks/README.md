# NyxCode Benchmarks

Measured, reproducible comparisons against popular alternatives. Every
benchmark ships with both source projects committed — you can check our
math by running the same commands.

## Available benchmarks

- [`blog/`](./blog/) — Full-stack blog with auth (NyxCode vs Next.js 14 + Prisma + JWT)

## Running them yourself

Each benchmark has its own README with reproduction steps. Results live in
`<benchmark>/RESULTS.md`.

We only publish benchmarks we've actually run. If a claim in the main README
doesn't have a folder here, don't trust it yet. 🦞
